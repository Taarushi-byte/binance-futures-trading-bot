# Binance Futures Testnet Trading Bot

## Overview
This is a simple Python trading bot that places MARKET and LIMIT orders on Binance Futures Testnet.

## Features
- Place MARKET orders
- Place LIMIT orders
- CLI input validation
- Structured code
- Logging of API requests and responses
- Error handling
